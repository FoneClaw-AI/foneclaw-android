# v0.0.4 Release Notes (Planned)

**Status:** Planned
**Target:** TBD

## Planned Features

- Self-growing Skill import from pasted `SKILL.md` text.
- GitHub-safe one-click Skill install links with `https://www.foneclaw.ai/skill/install?url=...`.
- Android deep link handling for `foneclaw://skill/import?url=...` and HTTPS install links.
- Digital Asset Links metadata for verified Android App Links.
- Skill preview with execution flow, referenced tools, validation status, and current state.
- User Skill lifecycle: save draft, enable, disable, inspect, and remove.
- Daily Device Brief sample Skill for validating the first custom Skill flow.

## Notes

This version focuses on making custom Skills safe to add and easy to inspect.
Only enabled and valid user Skills are injected into the agent prompt.
