# v0.0.2 Release Notes

**Release date:** June 17, 2026
**Dev cycle:** June 11 – June 17, 2026
**Version code:** 2
**APK size:** ~15.3 MB

## New Features

### Voice Input
Long-press the microphone button to talk. FoneClaw transcribes your speech and sends it as a message. Features a full-screen recording overlay with animation and cancel support.

### Intelligent Suggestion Chips
After each agent reply, 3–5 context-aware suggestion chips appear. New conversations show onboarding suggestions to help users discover capabilities.

### Google Play Auto-Install
FoneClaw can now open Google Play Store, find the install button, and click it — then wait for installation to complete via Accessibility Service. AI identifies button semantics and blocks paid/subscription actions automatically. Dual detection via package manager broadcast and notification listener.

### Notification Summary
Say "Summarize my notifications" and FoneClaw captures the notification bar and produces a structured summary organized by priority: need reply, need view/confirm, and other. Supports both Chinese and English.

### Shortcut Command System
Three types of shortcuts: Agent prompt shortcuts (skip intent parsing), direct tool shortcuts (skip LLM entirely), and auto-recommendation after task completion. Shortcuts are categorized and manageable.

### Homepage UX Optimization
New bottom bar, fixed layout issues (empty conversation overlap, keyboard interaction, scroll area), and enhanced Markdown/JSON card rendering.

### Prompt & Stability
English default prompts, hidden internal reasoning steps, cleaner tool output formatting, and more stable response format with fallback handling.

### Backend API Security
HMAC-SHA256 request signing with unique timestamp, nonce, and signature to prevent replay attacks.

## Known Issues

- Limited error recovery — no automatic retry on tool failure
- Model tuning still in progress
- Suggestion chip language detection inaccurate in mixed-language conversations
- Google Play auto-install requires Accessibility Service enabled
- Shortcut auto-recommendation is conservative
- Domestic (Chinese) manufacturer compatibility not yet implemented
- Third-party model integration pending

## Next Steps (v0.0.3)

- Firebase analytics integration
- Extended language support (Japanese, Korean, Traditional Chinese)
- Memory system optimization
